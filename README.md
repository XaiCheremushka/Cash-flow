# Веб-сервис для управления движением денежных средств (ДДС)
## Серверная часть веб-сервиса "Cash Flow"

**ДДС (движение денежных средств)** — это процесс учета, управления и анализа поступлений и списаний денежных средств компании или частного лица.

---
# Utiles

Основа: Python 3.13, Django, Django REST Framework
СУБД: PostgreSQL
Сервер: Gunicorn, Nginx

---
## Запуск приложения

1. Создать файл ".env" в корне проекта и заполнить по файлу-примеру ".env_example" значения переменных
2. Запустить из корня проекта docker-compose:
```bash
docker compose up -d --build
```
